### ETL  

Данный репозиторий содержит материалы и инструкции по выполнению лабораторных работ по дисциплине ETL (Extract, Transform, Load).  
В рамках работ рассматриваются процессы извлечения, трансформации и загрузки данных, а также построение полноценных конвейеров обработки данных с использованием современных инструментов.

---

## Содержание лабораторных работ  

### 1.Установка и настройка ETL-инструментов. Создание конвейеров данных  

Изучение базовых принципов работы ETL-систем, настройка инструментов и создание первых конвейеров обработки данных.

---

### 2.Динамические соединения с базами данных  

Настройка подключения к различным источникам данных, использование параметров и переменных для гибкой работы с базами данных.

Ссылка на выполнение:  
[Перейти к лабораторной работе](./lab1_README.md)

---

### 3.Интеграция данных из нескольких источников  

Объединение данных из различных источников, их обработка, очистка и приведение к единому формату для дальнейшего анализа.

Ссылка на выполнение:  
[Перейти к лабораторной работе](./lab1_README.md)

---

### 4.Анализ данных с использованием Dask и визуализация DAG  

Применение библиотеки Dask для обработки больших объемов данных, изучение принципов ленивых вычислений и визуализация направленных ациклических графов (DAG).

Ссылка на выполнение:  
[Перейти к лабораторной работе](./lab1_README.md)
